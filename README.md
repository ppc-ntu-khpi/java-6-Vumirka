## 🌸 Опис завдання

> Знайти **друге за величиною унікальне число** у двовимірній матриці розміром **N × M**
> без використання явних циклів — через клас `Arrays` та **Stream API**.

---

## 🧩 Приклад

### Вхідна матриця (3 × 3):

```
3   7   1
9   2  15
6   4   8
```

Унікальні значення за зростанням: `1, 2, 3, 4, 6, 7, 8, 9, 15`

**Друге за величиною → `9`** ✅
